# 抖音带货达人采集工具

这是一个用于学习网络爬虫、异步请求、数据清洗和桌面 GUI 开发的 Python 项目。项目围绕抖音 Web 端公开搜索和用户主页数据，完成从关键词搜索、达人提取、作品扫描、带货特征判断到 Excel 导出的完整实验流程。

本项目作者开发该工具的目的仅是作为网络爬虫学习作品和阶段性能力检验，用于理解 HTTP 请求、Cookie 登录态、接口参数、异步并发、限流重试、数据分析和本地报表生成等工程实践。

## 项目功能

| 功能 | 说明 |
| --- | --- |
| 关键词搜索 | 输入话题或关键词，采集相关视频搜索结果 |
| 达人提取 | 从视频结果中提取作者信息，按 `sec_uid` 去重 |
| 粉丝过滤 | 可设置最低粉丝数门槛，减少无效扫描 |
| 作品扫描 | 拉取达人主页最近作品，检查是否存在视频带货特征 |
| 带货判断 | 结合主页列表和作品详情信息判断是否为带货达人 |
| Excel 导出 | 将达人主页、昵称、带货视频链接、最近发布时间等字段导出为 `.xlsx` |
| GUI 操作 | 提供 Tkinter 桌面界面，支持配置参数、查看进度和导出结果 |
| Cookie 获取 | 支持浏览器登录提取、rookiepy 提取、CDP 提取和剪贴板解析 |
| 速度控制 | 内置安全、快速、极速三档请求间隔策略，降低触发限流的概率 |
| 多轮搜索 | 可使用不同排序方式多轮搜索后合并去重，提高样本覆盖率 |

## Cookie 配置

项目不会提交真实 Cookie。请复制 `main/.env.example` 为 `main/.env`，再填写自己的登录 Cookie：

```bash
cd main
copy .env.example .env
```

`.env` 示例：

```env
DY_COOKIE='your_douyin_cookie_here'
```

获取 Cookie 的常见方式：

1. 使用 Chrome 或 Edge 登录 `https://www.douyin.com/`。
2. 按 `F12` 打开开发者工具，进入 Network 标签页。
3. 刷新页面。
4. 右键任意请求，选择 `Copy` -> `Copy as cURL`。
5. 在 GUI 中点击粘贴，或手动从请求中提取 `Cookie` 字段写入 `.env`。

请妥善保管 Cookie。Cookie 等同于登录凭证，不要上传到 GitHub，不要发给他人。

## 使用方式

### GUI 方式

```bash
cd main
python main.py --gui
```

启动后可在界面中完成以下操作：

1. 获取或检测 Cookie。
2. 输入话题关键词。
3. 设置扫描作品数、搜索上限、最低粉丝数、排序方式、发布时间范围、并发数和速度模式。
4. 点击开始采集。
5. 采集完成后导出 Excel。

### 命令行方式

```bash
cd main
python main.py -k "防晒"
```

常用参数：

```bash
python main.py -k "美妆" -n 20 -s 1 -t 180 --max-search 200 --min-followers 10000
```

参数说明：

| 参数 | 简写 | 说明 |
| --- | --- | --- |
| `--keyword` | `-k` | 搜索关键词 |
| `--count` | `-n` | 每位达人扫描的作品数量 |
| `--sort` | `-s` | 排序方式，`0` 综合，`1` 最多点赞，`2` 最新发布 |
| `--time` | `-t` | 发布时间过滤，`0` 不限，`7` 一周内，`180` 半年内 |
| `--max-search` | | 搜索结果上限 |
| `--min-followers` | | 最低粉丝数过滤 |
| `--output` | `-o` | Excel 输出目录 |
| `--cookie` | `-c` | 直接传入 Cookie 字符串 |
| `--gui` | | 启动桌面 GUI |

## 输出结果

默认输出目录为 `main/output/`，文件名格式如下：

```text
{关键词}_视频带货达人_{时间戳}.xlsx
```

Excel 字段包括：

| 字段 | 说明 |
| --- | --- |
| 达人主页链接 | 可在浏览器中打开的达人主页 URL |
| 昵称 | 达人昵称 |
| 最近挂车作品链接 | 最近检测到的带货视频链接 |
| 最后一条视频发布时间 | 达人最近作品发布时间 |

## 速度和风控建议

项目内置三档速度策略：

| 模式 | 特点 | 建议场景 |
| --- | --- | --- |
| 安全 | 请求间隔较长，稳定性优先 | 首次测试、长时间运行 |
| 快速 | 适当缩短请求间隔 | 日常小规模采集 |
| 极速 | 间隔最短，风险最高 | 少量样本快速验证 |

建议先从安全模式和较低并发开始。如果出现请求失败、HTTP 444、结果为空或账号异常提示，应立即降低频率、减少并发并暂停使用。

## 常见问题

### Cookie 无效怎么办

重新登录抖音 Web 端并更新 `main/.env`。Cookie 可能会过期，也可能因为环境变化被平台判定为无效。

### 搜索不到数据怎么办

检查关键词是否过窄，确认 Cookie 有效，降低速度模式和并发数后重试。

### 为什么部分达人判断不准确

工具只扫描最近若干条作品，并依赖 Web 端接口返回字段进行判断。若作品数量设置过低、接口字段变化、达人隐藏作品或平台策略变化，都可能导致误判。
